üìù What Happens Next

After clicking "Login with Google":

  • You'll be redirected to: https://accounts.google.com/oauth/authorize?...
  • Google will ask you to sign in and grant permissions
  • Google will redirect you back to: http://localhost:8000/accounts/google/login/callback/
  • Django will process the OAuth response and create/login the user
  • You'll be redirected to: http://localhost:8000/auth/success/
  • Check your Django server terminal for logs

üîç Debugging

If you get errors, check:

  • Django server is running on http://localhost:8000
  • Google OAuth credentials are correctly set in .env
  • Google Cloud Console has the correct redirect URI: http://localhost:8000/accounts/google/login/callback/
  • Your Google project has the Google+ API enabled
